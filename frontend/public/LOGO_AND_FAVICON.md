# Logo and favicon

Add your MedAI branding assets here:

- **`logo.png`** – Used as the navbar logo, footer logo, and on auth pages. Prefer a square or near-square image (e.g. 80×80 or 160×160 px) for best results; aspect ratio is preserved.
- **`favicon.ico`** – Browser tab and bookmarks. Standard sizes: 16×16, 32×32, 48×48.
- **`apple-touch-icon.png`** (optional) – 180×180 px for iOS home screen.

If `logo.png` is missing, the app shows the default Heart icon so the UI still works.
